
# nmap_polygon_uploader
*** 

**Скрипт для переноса геометрии shapefile в «Блокнот картографа» для его дальнейшего нанесения.**
<br>
<br>

### **Как работает**

После запуска он анализирует архив и записывает в файл index.json информацию о внешних (exterior) и внутренних (interior) границах геометрии.
В разделе «Народная карта» на вашем Яндекс Диске создаётся папка с названием текущей даты. В эту папку автоматически загружается файл index.json. После завершения загрузки файлы удаляются. Необходимо обновить страницу редактора и выбрать в блокноте нужный файл с загруженными заметками.

_**Этот скрипт работает с шейпфайлами от ОЦЗК**. Его можно использовать и с другими шейпфайлами, если архив назвать `geoserver_data.zip`, а вложенные файлы — `oopt_wth_detailsPolygon`._

### **Данные**

GIS [оцзк.рф](https://карта.оцзк.рф/) помогает найти данные об особо охраняемых природных территориях (ООПТ) в России. С её помощью можно скачать информацию о таком объекте.

Информация представлена в виде архива с файлами Shapefiles. В архиве содержатся сведения о местоположении, конфигурации и характеристиках географических объектов. С помощью скрипта nmap_polygon_uploader происходит обработка архива и преобразование данных в формат, который может быть использован в редакторе народной карты.


### **Блокнот картографа**

Подробнее о приложении и веб версии данного инструменте можно [прочитать на страницах справки яндекса](https://yandex.ru/support/nmaps/ru/map-blocknot.html) и в архивном [посте](https://yandex.ru/blog/narod-karta/bloknot-kartografa) клуба народной карты (закрыт). 


### **Как запустить**

1. Необходимо переименовать файл `.env.example` --> `.env`
2. В файле .env для переменной ` ZIP_PATH ` указать дефолтный путь сохранения файлов
3. В файле .env для переменной ` YD_TOKEN ` указать ваш OAuth-токен Яндекс ([получить](https://yandex.ru/dev/disk/poligon/))
